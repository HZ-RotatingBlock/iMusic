# iMusic

一个基于 `React` 与 `electron` 开发的音乐App。

感谢 [NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi) 提供的 API。

## 项目结构

```
├── config // 项目打包配置
├── mock // 模拟数据
├── src
│    ├── actions
│    ├── api // 请求接口
│    ├── assets // 公用静态资源
│    ├── components // (Redux)组件
│    ├── containers // (Redux)容器
│    └── reducers
├── resource // 后期可能调整该目录
├── dist
└── release
```
